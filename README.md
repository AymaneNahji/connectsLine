## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd connectsline
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run tailwind
```
